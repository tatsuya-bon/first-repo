# first-repo

このプロジェクトは、Git と GitHub の練習のためのリポジトリです。

## できること

- ファイルの追加や変更を試せます
- コミットの練習ができます
- ブランチの作成や変更を試せます
- Pull Request の流れを確認できます

## 入っているファイル

- `readme.txt`: 練習用のテキストが入っています

## 使い方

1. ファイルを編集します
2. 変更をコミットします
3. 必要ならブランチを作ります
4. GitHub に push して確認します

## 補足

このリポジトリは学習用なので、内容はシンプルです。
Git の基本操作に慣れるために使えます。
